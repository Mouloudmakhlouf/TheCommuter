# TheCommuter

TheCommuter is an interdisciplinary project combining Computer Science and Microeconomics to calculate dynamic commuting costs using live toll and transportation data.

## Project Overview

The application estimates the real cost of commuting by analyzing:

- Live toll pricing
- Fuel consumption
- Route distance
- Transportation expenses

This project demonstrates how software engineering and economic analysis can work together to improve transportation decision-making.

---

## Features

- Dynamic commute cost calculation
- Toll route analysis
- Fuel expense estimation
- Interactive user interface
- Responsive web design

---

## Technologies Used

- HTML5

## Interdisciplinary Focus

This project combines:
